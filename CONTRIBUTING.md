# 贡献指南

感谢你对这个项目的兴趣！我们欢迎所有形式的贡献。

## 🤝 如何贡献

### 报告 Bug

如果你发现了 bug，请：

1. 检查 [Issues](https://github.com/yourusername/openai-chat-demo/issues) 确认问题还没有被报告
2. 创建一个新的 Issue，包含：
   - 清晰的标题和描述
   - 重现步骤
   - 预期行为和实际行为
   - 你的环境信息（Node.js 版本等）

### 建议新功能

1. 先创建一个 Issue 讨论你的想法
2. 等待维护者的反馈
3. 如果获得同意，你可以开始实现

### 提交代码

1. **Fork** 这个仓库
2. **Clone** 你的 fork：
   ```bash
   git clone https://github.com/yourusername/openai-chat-demo.git
   ```
3. 创建一个新分支：
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. 进行你的更改
5. 测试你的更改：
   ```bash
   npm test
   ```
6. 提交你的更改：
   ```bash
   git commit -m "Add: 你的功能描述"
   ```
7. 推送到你的分支：
   ```bash
   git push origin feature/your-feature-name
   ```
8. 创建一个 **Pull Request**

## 📝 代码规范

- 使用有意义的变量和函数名
- 添加必要的注释
- 保持代码简洁易读
- 遵循现有的代码风格

## 🧪 测试

在提交 PR 之前，请确保：

- 你的代码能正常运行
- 没有破坏现有功能
- 添加了必要的测试（如果适用）

## 📋 提交信息格式

请使用清晰的提交信息：

- `Add: 添加新功能`
- `Fix: 修复某个问题`
- `Update: 更新某个功能`
- `Docs: 更新文档`
- `Style: 代码格式调整`

## 🙋‍♀️ 需要帮助？

如果你在贡献过程中遇到任何问题，请：

1. 查看现有的 Issues 和 Pull Requests
2. 创建一个新的 Issue 寻求帮助
3. 发送邮件到：your.email@example.com

## 🎉 感谢

感谢所有贡献者的努力！你们的贡献让这个项目变得更好。
